A document management system developed to help organizations manage quality-related documents in a more structured and systematic way. It supports better document storage, tracking, and organization.

A smart training tool designed to test and improve athlete reaction time. This project focuses on measuring response speed and supporting sports training performance through interactive testing.

A smart QR-based vehicle access system for university vehicle management. The system allows users to register vehicles, generate QR stickers, and support vehicle verification for campus security access control.
